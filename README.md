  | Option 参数 | Remark 备注 | 
  | -----------| ------ |
  | -c         | Chinese 中文 |
  | -e         | English 英文 | 
  | -f         | Variable file，refer to REPO file "config" 参数文件，可参数项目的文件 config | 
  | -u         | Uninstall 卸载 |
  | -e         | Export Node list 显示节点信息 |
  | -v         | Sync Argo Xray to the newest 同步 Argo Xray 到最新版本 |
